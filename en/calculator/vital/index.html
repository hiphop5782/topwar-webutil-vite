What Is the Vitality Recovery Speed Calculator?

This tool calculates the actual time required to recover 1 vitality using the Vitality Recovery Speed and Base Vitality Recovery Speed shown in Base Information.

Enter a decoration's Base Vitality Recovery Speed bonus to compare the recovery interval, time reduction, improvement percentage, and estimated recovery amount before and after applying it.

How to Use the Calculator

  1. Open Base Information in the game and locate Vitality Recovery Speed and Base Vitality Recovery Speed.
  2. Enter both values exactly as displayed in the first input section.
  3. Enter the Base Vitality Recovery Speed bonus of the decoration you plan to purchase or place.
  4. Review the recovery time and estimated hourly and daily recovery before and after applying the decoration.

Calculation Method

The calculation starts from a base recovery time of 360 seconds and applies the relationship between total recovery speed and base recovery speed.

Current Recovery Time
((100 - Vitality Recovery Speed + Base Vitality Recovery Speed) ÷ (100 + Base Vitality Recovery Speed)) × 360 seconds
Recovery Time After Decoration
((100 - (Vitality Recovery Speed + Decoration Bonus) + (Base Vitality Recovery Speed + Decoration Bonus)) ÷ (100 + Base Vitality Recovery Speed + Decoration Bonus)) × 360 seconds

Important Notes

  • Enter the percentage values exactly as displayed in the game.
  • For better accuracy, do not round values that include decimal places.
  • Hourly and daily recovery amounts assume that vitality never reaches its maximum capacity.
  • Actual results may differ if the recovery formula changes in a future game update.

Frequently Asked Questions

Vitality Recovery Speed includes the currently applied total bonuses, while Base Vitality Recovery Speed is a separate value used by the formula. Both values are required for an accurate result.

Enter the Base Vitality Recovery Speed percentage shown in the decoration details. Do not confuse it with a general Vitality Recovery Speed bonus.

Even a small difference in the percentage can change the recovery interval by several seconds. Enter the displayed decimal value whenever available.
