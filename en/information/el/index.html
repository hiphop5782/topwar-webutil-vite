Calculation Basis

Hourly scores are calculated as 60 times each facility's base per-minute score, and daily scores as 1,440 times the base score.

The result is a simple estimate that assumes the selected facilities remain occupied for the entire period.

Important Limitations

Actual scores may vary depending on the occupation start time, facility loss, server progress, and event-rule changes.

Please contact the site if the values have changed after a game update.

Frequently Asked Questions

Can I select multiple facilities at once?

Yes. Select facility markers one by one, and the scores of all selected facilities will be added automatically.

Will I actually earn the full daily score shown?

The daily score assumes continuous occupation for 24 hours. The actual score will be lower if the occupation period is shorter or the facility is lost.

What if a facility location differs from the game map?

Facility layouts and values may change by Eternal Land season or game update. Use the latest in-game map as the primary reference.
