Frequently Asked Questions

Is this data updated in real time?

No. The page searches a player dataset stored by the site, so it may differ from the game's current state.

Can I search by server and nickname together?

Yes. When both fields are entered, the viewer shows only players on that server whose nickname contains the entered text.

Why does the rank remain the global rank after filtering?

Rank is calculated from combat power across the full dataset before filters are applied. Search results are not ranked again.
