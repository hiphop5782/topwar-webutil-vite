What Is the Titan Gear Crafting Simulator?

This tool generates possible Titan gear grades, standard options, and special effects using probability-based simulation.

The selected part, catalyst, and craft count are stored in the URL, making it easy to share or revisit the same settings.

How to Use the Simulator

  1. Select the Titan gear part you want to craft.
  2. Choose a catalyst or select No Catalyst.
  3. Enter a craft count between 1 and 100.
  4. Run the simulator and review the grade distribution, option values, and special effects.

Simulation Method

Gear Grade

One grade is selected from Green, Blue, Purple, and Gold using the rate table for the selected catalyst.

Standard Options

Each gear receives one or two options at a 40% chance each, or three options at a 20% chance. Option values use a normal distribution so values near the midpoint occur more often.

Special Effects

Special effects are rolled only for Gold gear and use the rate table assigned to the selected gear part.

Important Notes

  • Results are generated locally in the browser and do not guarantee actual in-game crafting outcomes.
  • The option-value distribution uses this simulator's normal-distribution model and may differ from the game's internal algorithm.
  • Results may become outdated if a game update changes probability tables or option ranges.
  • Small samples can differ greatly from the configured rates; larger samples tend to approach the expected distribution.
