TOP WAR 시뮬레이터

타이탄 재련 시뮬레이터

타이탄 장비와 재료 옵션을 직접 설정하고, 재련 시 옵션 추가·수치 상승·덮어쓰기 결과를 시뮬레이션할 수 있습니다.

재련 부위 선택

부위를 변경하면 장비, 재료, 기록이 해당 부위의 기본값으로 초기화됩니다.

타이탄 장비 설정

옵션은 1개부터 3개까지 설정할 수 있으며, 각 옵션을 눌러 종류와 수치를 변경할 수 있습니다.

돌격 권총

2번 빈 옵션 슬롯
3번 빈 옵션 슬롯

재련 재료 설정

재료 장비가 제공할 옵션 종류와 수치를 설정합니다.

돌격 권총

재련 시 가능한 옵션 슬롯 중 하나가 무작위로 선택됩니다. 같은 옵션이면 수치가 상승하고, 다른 옵션이면 재료 옵션으로 교체됩니다.

타이탄 재련 시뮬레이터란?

타이탄 장비의 일반 옵션과 재련 재료 옵션을 직접 구성한 뒤, 재련 결과가 어떤 방식으로 적용되는지 확인하는 도구입니다.

기존 코드의 재련 규칙과 JSON 데이터에 등록된 옵션 최대치·재련 상승 범위를 사용하며, 결과와 기록은 브라우저에서만 계산됩니다.

재련 규칙

  1. 장비에 빈 슬롯이 있으면 빈 슬롯도 재련 대상에 포함될 수 있습니다.
  2. 선택된 슬롯의 옵션과 재료 옵션이 같으면 JSON에 등록된 상승 범위만큼 수치가 증가합니다.
  3. 선택된 슬롯의 옵션과 재료 옵션이 다르면 기존 옵션이 재료 옵션으로 교체됩니다.
  4. 같은 종류의 옵션이 이미 최대 수치인 슬롯은 재련 대상에서 제외됩니다.

자주 묻는 질문

재료 옵션과 같은 종류이면서 이미 최대 수치인 슬롯은 더 이상 상승할 수 없으므로 재련 대상에서 제외됩니다.

기존 옵션은 유지되지 않고 재료 장비의 옵션 종류와 수치로 교체됩니다.

등록된 옵션 범위와 재련 상승 데이터를 기반으로 하지만 실제 게임의 내부 난수와 규칙이 변경될 수 있으므로 참고용으로 사용해야 합니다.
