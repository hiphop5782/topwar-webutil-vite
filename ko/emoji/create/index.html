무료 AI 이미지 배경 제거 및 정사각형 편집기

사진의 배경을 자동으로 제거하고, 잘못 제거된 부분을 브러시로 직접 지우거나 복구한 뒤 정사각형 PNG 이미지로 저장할 수 있습니다.

프로필 이미지, 게임 커뮤니티 이모티콘, 상품 이미지처럼 배경이 투명한 1:1 이미지를 만들 때 사용할 수 있습니다.

1단계: 이미지 선택

배경을 제거할 PNG, JPG, JPEG 또는 WebP 이미지를 선택하세요. 첫 실행은 AI 모델을 준비하느라 조금 더 오래 걸릴 수 있습니다.

선택한 이미지 파일은 이 도구의 배경 제거 작업을 위해 브라우저에서 처리되며, 이 페이지의 코드에서는 이미지 파일을 별도 서버로 업로드하지 않습니다.

이 도구에서 할 수 있는 작업

자동 배경 제거

이미지 속 주요 대상을 분석하여 배경을 투명하게 만듭니다.

수동 지우기와 복구

자동 처리에서 남은 배경은 지우고, 사라진 대상 부분은 다시 복구할 수 있습니다.

정사각형 PNG 저장

위치와 확대 비율을 조절한 뒤 투명 배경을 유지한 PNG 파일로 저장합니다.

사용 방법

  1. 배경을 제거할 이미지 파일을 선택합니다.
  2. 자동 분석이 끝나면 대상의 가장자리와 투명 처리 결과를 확인합니다.
  3. 왼쪽 클릭으로 불필요한 부분을 지우고, 오른쪽 클릭으로 잘못 지운 부분을 복구합니다.
  4. Ctrl 키와 마우스 휠로 확대·축소하고, 마우스 휠 버튼을 누른 채 이미지를 이동합니다.
  5. 원하는 구도가 완성되면 정사각형 PNG 저장 버튼을 누릅니다.

이미지 처리와 개인정보 안내

이 페이지는 @imgly/background-removal 라이브러리를 사용해 사용자의 브라우저에서 배경 제거를 실행합니다. 현재 구현에는 선택한 이미지 파일을 Progamer.info 서버로 전송하거나 저장하는 업로드 코드가 없습니다.

다만 사이트 운영에 필요한 일반적인 접속 로그, 쿠키 및 분석 도구의 처리 여부는 사이트 개인정보처리방침을 함께 확인해 주세요.

결과가 정확하지 않을 수 있는 경우

  • 머리카락, 털, 반투명 소재처럼 경계가 매우 복잡한 이미지
  • 대상과 배경의 색상 또는 밝기 차이가 거의 없는 이미지
  • 해상도가 너무 낮거나 심하게 흔들린 이미지
  • 한 장에 여러 대상이 겹쳐 있어 주요 대상을 구분하기 어려운 이미지

자동 처리 결과는 보조 기능이므로, 중요한 이미지라면 저장 전에 가장자리를 확대해 직접 확인하세요.

저작권 및 이용 시 주의사항

본인이 촬영하거나 편집 권한을 가진 이미지만 사용하세요. 타인의 사진, 캐릭터, 상표 또는 저작물을 편집·배포할 경우에는 해당 권리자의 이용 조건을 확인해야 합니다.

자주 묻는 질문

이 페이지의 배경 제거 작업은 브라우저에서 실행되며, 현재 컴포넌트에는 이미지 파일을 별도 서버로 업로드하거나 저장하는 코드가 없습니다.

브라우저가 배경 제거에 필요한 AI 모델과 실행 파일을 준비해야 하기 때문입니다. 이후 실행은 브라우저 캐시와 기기 상태에 따라 더 빨라질 수 있습니다.

일반적으로 PNG, JPG, JPEG, WebP 형식을 사용할 수 있습니다. 브라우저가 해당 이미지 형식을 해석할 수 있어야 합니다.

네. 저장 형식은 투명 배경을 지원하는 PNG이며, 편집 화면의 1:1 정사각형 영역을 기준으로 저장됩니다.

이미지 선택과 자동 배경 제거는 가능할 수 있지만, 현재 수동 편집 조작은 마우스를 기준으로 설계되어 데스크톱 브라우저 사용을 권장합니다.
