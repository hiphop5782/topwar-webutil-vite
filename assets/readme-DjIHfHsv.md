메카는 유닛 슬롯 한 칸을 차지하며 부대에게 각종 효과를 부여하고 스킬을 사용한다.  
메카의 유무는 승패를 좌우하는 가장 중요한 요소 중 하나이다.

![블레이드 엔젤 메카](./thumbnail.gif)

메카는 일일 임무 중 **미지의 작전** 메뉴에서 확인할 수 있다.

![미지의 작전 메뉴](./001.png)

![미지의 작전 화면](./002.png)

# 메카 종류
메카는 작성 시점(2024-10-17) 기준 총 6종류가 출시되었다.

- 썬더볼트-C7
- 질풍의불꽃-Rx
- 파도 분쇄자 BW-3
- 파멸의 톱날 D-4
- 광휘의 기사 LP-4
- 블레이드 엔젤 BA-6

각 메카별 특징은 다음과 같다.

| 메카 | 분류 | 특징 |
| --- | --- | --- |
| 썬더볼트-C7 | 방어형 | 면역 스킬 |
| 질풍의불꽃-Rx | 공격형 | 펄스의 영향을 받지 않음 |
| 파도 분쇄자 BW-3 | 밸런스형 | 기절 및 후방 데미지 증가/감면 |
| 파멸의 톱날 D-4 | 밸런스형 | 자기장 및 중장갑 |
| 광휘의 기사 LP-4 | 공격형 | 크리스 칩과 시너지 효과 |
| 블레이드 엔젤 BA-6 | 공격형 | 메카 수리 1회 및 중/경장갑 특화 <span style="color:red">(과금으로만 획득 가능)</span> |

# 메카 칩

메카 칩은 7개 부위로 나눠진다.

![메카 슬롯 구분](./003.png)

| 종류 | 옵션 개수 | 특징 |
| --- | --- | --- |
| 코어 | 1~2개 | 모든 옵션이 나올 수 있는 칩 <span style="color:red">(강화 불가)</span> |
| 1, 2, 3번 | 1~4개 | **메카 본체**에 대한 옵션을 보유한 칩 <span style="color:blue">(강화 가능)</span> |
| 4, 5, 6번 | 1~4개 | **유닛 군종**에 대한 옵션을 보유한 칩 <span style="color:blue">(강화 가능)</span> |

메카 칩은 현재까지 출시된 탑워 아이템 중에서 가장 많은 **방어도**를 획득할 수 있기 때문에 고려해서 세팅해야 한다.

## 종류

메카 칩은 다음과 같은 종류가 존재한다.

| 구분 | 이름 | 특징 |
| --- | --- | --- |
| 공격 | 자이스톤 | 생명력 60% 이상 목표 공격 시 메카 데미지 +30% |
| | 샴시르 | 매 라운드 메카로 입힌 데미지 8%씩 증가 (최대 48%) |
| | 스크램 | **점화** 상태의 유닛 공격 시 데미지 35% 증가 |
| | 론델 | 40% 확률로 일반 공격 1회 추가 (치명타 시 100%) |
| | 크리스 | 매 라운드 메카 일반공격 1회당 메카 데미지 8%씩 증가(최대 32%) |
| | 건곤권 | 메카가 보유한 데미지 감쇄 실드 1종류마다 메카 데미지 18% 증가(최대 36%) |
| 방어 | 호플론 | 메카 생명력 증가 15% (맨 앞줄 배치시 +10%) |
| | 에스톡 | 메카를 공격한 상대에게 80% 확률로 30% **쇠약** 추가 |
| | 사사니안 | 메카 생명력 증가 18%, **[면역/중장갑/경장갑]** 스택 +2 |

코어 칩에 따라 옵션이 달라지며, 부동의 1티어는 **자이스톤**이다.  
메카 또는 영웅의 조합에 따라 다른 칩을 사용할 수도 있다.  
세트 효과가 있으나 옵션이 좋다면 세트 효과를 포기하고 사용할 수도 있다.

[[TIP]]
2024년 10월 출시된 **건곤권** 칩은 풀 세트 장착 시 데미지 증가 30%가 추가된다.  
[[/TIP]]

## 코어 칩
코어 칩은 모든 옵션이 1~2개 존재할 수 있다.
가장 좋은 옵션으로 분류되는 것들은 다음과 같다.

1. 전체 방어도 + 육/해/공 방어도
2. 전체 방어도 + 전체 데미지 증가/감면
3. 전체 데미지 증가 + 전체 데미지 감면 
4. 전체 데미지 증가/감면 + 육/해/공 데미지 증가/감면

![메카 코어칩 옵션](./004.png)

[[TIP]]
수치는 칩의 등급에 따라 다르며 최대 `33.xx%`까지 나오는 것으로 알려져 있다.
[[/TIP]]

## 1~3번 칩

1,2,3번 칩은 메카 관련 옵션을 가지며은 크게 두 가지 컨셉으로 나눠진다.

- 공격속도와 데미지 관련 옵션 칩
- 데미지와 방어 관련 옵션 칩

각 컨셉별로 중요한 옵션은 다음과 같다.

| 컨셉 | 주요 옵션 |
| --- | --- |
| 데미지 우선 | <span style="color:red">메카 공격 속도</span><br><span style="color:red">메카 데미지 증가</span><br>메카 데미지 감면<br>메카 치명타 데미지 증가<br>메카 공격력<br>메카 생명력 |
| 밸런스 우선 | <span style="color:red">메카 데미지 증가</span><br><span style="color:red">메카 데미지 감면</span><br><span style="color:red">메카 실드 획득</span><br><span style="color:red">메카 방어도</span><br>메카 공격력<br>메카 생명력 |

[[TIP]]
과거 약한 유저들이 많았을 때는 먼저 공격하는 것이 중요해서 공격속도에 올인하고 나머지 옵션을 포기하는 경향이 강했으나 시간이 흘러가면서 한번의 공격으로 없앨 수 있는 유닛의 숫자가 줄어들며 선제공격은 의미가 점점 사라지고 있다.
[[/TIP]]

## 4~6번 칩

4,5,6번 칩은 유닛 군종 관련 옵션을 가지며 대체로 중요한 옵션들이 정해져 있다.

| 주요 옵션 |
| --- |
| <span style="color:red">전체 방어도 증가</span> <br> 전체 데미지 증가<br>전체 데미지 감면<br><span style="color:red">육해공 방어도 증가</span><br>육해공 데미지 증가<br>육해공 데미지 감면 |


# 칩 오버클럭

에픽 메카 칩은 오버클럭해서 더 좋은 옵션을 가질 수 있도록 만들 수 있다.

![오버클럭 과정](./005.png)

- 칩 오버클럭 - 일반 에픽 칩 4개로 자수정 칩 1개를 제작 (옵션 및 종류 선택 불가)
- 고급 칩 오버클럭 - 자수정 칩 4개로 오팔 칩 1개를 제작 (군종 선택 가능, 옵션 선택 불가)
- 탑 오버클럭 - 오팔 칩 4개로 루비 칩 1개를 제작 (군종 및 옵션 선택 가능)

탑 오버클럭을 하기 위해서는 일반 에픽 칩 64개가 필요하며, 이를 위해 꾸준하게 주간 카드를 구매하는 것을 추천한다.  
특히 주간 카드 상점 레벨에 `6`이 되면 에픽칩 보상이 `2`개로 늘어나므로 어느 정도 과금을 하는 유저라면 이 구간을 노려볼 것을 권장한다.

![주간 커스텀 카드](./006.png)


오버클럭은 다음과 같은 순서로 진행하는 것을 추천한다.

1. 코어칩 - 고급 or 탑 오버클럭 (방어도 옵션 우선)
2. 4~6번칩 - 고급 or 탑 오버클럭 (방어도 옵션 우선)

1,2,3번은 오버클럭보다 일반 칩으로 맞추고 여력이 된다면 2군의 코어 칩부터 다시 오버클럭하는 것이 더 효율적이다.
