# Topwar의 데미지 계산 공식

공식적으로 Topwar에서는 데미지 계산 공식을 공개한 적이 없습니다.  
일반적인 게임의 계산 방식과 과거 가장 게임사의 정보를 많이 소개하던 유튜버 `T-me`와 `Nass`의 정보를 토대로 추리하여 작성하는 글임을 미리 밝힙니다.

탑워의 데미지 계산 방식에 대한 추측 공식은 다음과 같습니다.

```
최종 피해량
=
기초 공격력
× [1
   + 2 × (병종 특화 공격력% + 장비 공격력% + 부품 공격력%)
   + 영웅 WAR 공격력% × 1.3
   + 기타 공격력%
  ]
× [1
   + 일반 데미지 증가%
   + 영웅 데미지 증가%
   + 상성 데미지 증가%
  ]
× (1 + 영웅 공격력%)
× 타격 배율
× (1 + 취약%)
× 스택 수
÷ [1
   + 상대 일반 방어력 증가%
   + 상대 상성 방어력 증가%
  ]
× (1 - 파괴불가 데미지 감소%)
```

간단하게 살펴보면
- 공격자 : 유닛의 기본 공격 데미지에 공격자의 공격력(atk), 데미지증가(dmg+)를 더하거나 곱한다.
- 방어자 : 공격자의 데미지를 방어자의 데미지감면(dmg-)과 방어도(def)로 감소시킨 뒤 생명력에서 차감한다.

더 간단하게 살펴보면 다음과 같습니다.

```
최종데미지 = (공격 데미지 합계) - (방어로 감소된 데미지)
```

하지만 간단하게 살펴보면 어느것이 어떤 효율을 가지고 있는지 알 수 없기 때문에 각각의 항목들에 대해서 알아보겠습니다.
