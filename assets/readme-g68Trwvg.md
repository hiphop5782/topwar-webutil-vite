# 폭풍의 눈 전투

폭풍의 눈은 `봉인석의 난` 이벤트에서 가장 많은 랭킹포인트를 주는 전장입니다.  
1주일 점령 시 `200`점을 주기 때문에 공략을 숙지하여 점령 확률을 높이도록 노력합시다!

![폭풍의 눈 메뉴](<./폭풍의 눈 메뉴.png>)

# 전체 지도

![폭풍의 눈 지도](<./폭풍의 눈 지도.png>)

폭풍의 눈에는 총 `12`개의 외곽 유적과 `9`개의 내곽 유적이 존재합니다.  
편의상 시계방향으로 다음과 같이 부르도록 하겠습니다.  

![폭풍의 눈 지도2](<./폭풍의 눈 지도2.jpg>)


# 게임 타임라인
- 0분 : 입장 및 평화 상태
- <span style="color:red">5분 : 외곽 유적 개방 (S1 ~ S12)</span>
- 10분 : 수호수 출전
- 15분 : 채집 (봉인석 크리스탈 광맥)
- 20분 : 수호수 출전
- <span style="color:red">25분 : 내곽 유적 개방 (H1 ~ H9)</span>
- 28분 : 채집 (봉인석 크리스탈 광맥)
- 30분 : 수호수 출전
- <span style="color:red">35분 : 중앙 보스 공격 (초능력 수위병)</span>
- 40분 : 수호수 출전
- 42분 : 채집 (봉인석 크리스탈 광맥)
- <span style="color:red; font-weight:bold;">45분 : 전장 연소 단계 (3배 버닝 타임)</span>
- 50분 : 수호수 출전
- 60분 : 전투 종료

# 폭풍의 눈에서 점수를 얻는 방법

폭풍의 눈에서 점수를 얻는 방법은 다음과 같습니다.

1. 유적을 최초 점령하여 점수 획득
2. 유적을 지속 점령하여 점수 획득
3. 유적을 탈환하여 바닥에 떨어진 포인트 획득
4. 채집을 통한 점수 획득
5. 보스 공격 등수를 통한 점수 획득

이를 위해서는 지도를 보는 방법을 알아야 합니다.

## 소유자 변경 시

2. 정찰했을 때 자신이 직접 해결이 가능하다면 즉시 회복

# 시간대별 유적 현황

외곽 유적은 전투 개시인 `5분`부터, 내곽 유적은 `25분`부터 점령 가능 상태가 됩니다.
최초 점령 점수가 있기 때문에 시간을 잘 체크하고 공유해야 합니다.

- 외곽 유적 (`S-1` ~ `S-12`) : 개당 `500`점
- 내곽 유적 (`H-1` ~ `H-9` ) : 개당 `1000`점

# 자신에 전투력에 맞는 플레이 스타일

각 유저들은 전투력에 따라 다음과 같이 역할을 나눠야 합니다.

1. 높은 CP를 가진 인원 - 유적의 소유권 탈환 및 평화모드 만들기
2. 중간 CP를 가진 인원 - 병력이 적은 유적의 소유권 탈환 및 포인트 수집, 채집
3. 낮은 CP를 가진 인원 - 병력이 적은 유적의 소유권 탈환 및 포인트 수집, 채집
4. 모니터 인원 - 외곽 안전지대에서 지도 상황에 따른 채팅 알림

평화모드를 만들지 않더라도 소유권만 뺏으면 상대방은 즉시 지속포인트를 잃게 됩니다.  
이것만으로도 상대에게 큰 타격을 줄 수 있습니다.

# 이동 및 가속 주의사항

이동권은 최대 `3`개, 가속은 최대 `10`개까지만 보유할 수 있기 때문에 적절하게 계산해서 사용해야 합니다.  

- 이동권 회복 시간 : 4분 (최초 3개 + 회복 13개 = 총 16개)
- 가속권 회복 시간 : 1분 (최초 10개 + 회복 55개 = 총 65개)

점액의 시간이 매우 짧기 때문에 점액을 맞은 경우 무리하지 말고 기다려도 됩니다.

# 높은 CP 전투 인원들이 해야할 일

높은 CP를 보유한 유저들은 적진에 침투해서 상대 서버의 유적을 우리것으로 탈환해야 합니다.

# 낮은 CP 전투 인원들이 해야할 일

낮은 CP를 보유한 유저들은 채집을 하고 도움이 필요한 곳을 아군에게 알려야 합니다.  
또한 탈환된 유적을 지도에서 확인하고 바닥에 떨어진 포인트를 점령해야합니다.

1. 적이 점령중인 채집지를 공격해서 제거 후 채집

2. 지도에서 탈환 신호를 확인하면 아군에게 알리고 보상 포인트를 획득

# 보스 공격

전투 시작 `35분`이 되면 중앙에 보스 공격이 가능합니다.  

![alt text](./image-10.png)


공격을 위한 최적의 세팅은 다음과 같지만 전투 진행중에 칠 때는 신경쓰지 않고 횟수를 채우는데 집중해야 합니다.
- 파츠 : 공격속도 + 무기강화 + 데미지증가 + 치명타 + 치명타데미지증가
- 메카 : 1, 4, 7번 중 하나
- 군진 : 샤크 군진(추가 슬롯 때문에 1턴 더 버틸 수 있음)

보스 데미지 1등을 한 서버에게는 `10000`포인트가 추가로 주어집니다.

# 전장 연소 (3배 버닝 타임)

![alt text](<./전장 연소.gif>)

`45분`부터 종료 시점까지 전장 연소 단계에 돌입합니다.  
이 기간에는 특정 지역이 3배의 점수를 가지게 되어 단시간에 점수차이를 좁힐 수 있습니다.  

지도에 불특정 다수의 유적이 계속해서 강조 표시되며 이 지역을 점령하면 다른 유적대비 **3배**의 점수를 획득할 수 있습니다.  

![alt text](./image-11.png)

모든 인원들은 해당 지역을 같은 서버에게 알리거나, 탈환하는데 힘을 모아야 합니다.
